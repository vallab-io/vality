# 서비스 개요 (Overview)

## 🎯 서비스 핵심 컨셉

> "뉴스레터를 발행하면 곧바로 웹에 기록되고, 검색 엔진에도 노출되는 개인 브랜딩 플랫폼."

### 핵심 가치 제안 (Value Proposition)

| 구분 | 설명 | 효과 |
|------|------|------|
| 이메일 뉴스레터 | 콘텐츠 유통 채널 | 구독자와 직접 소통 |
| 웹 게시글 동시 노출 | 검색/SEO 확보 | 검색 유입을 통한 신규 독자 확보 |
| Bio | 링크 기반 개인·브랜드 명함 | 통합 브랜드 허브 (추후 개발) |

### 서비스 차별화 포인트

```
한 번 발행 → 이메일 · 블로그 · 프로필 콘텐츠가 동시에 완성
```

기존 서비스들은 각각의 목적에 특화되어 있습니다:

- **Substack, Buttondown** → 이메일 뉴스레터 전문
- **Medium, Tistory, Velog, Notion** → 블로그/아티클 전문
- **Linktree, Bento** → 링크 허브 전문

**Vality**는 이 세 가지를 하나로 통합하여:
1. 콘텐츠 제작자의 작업 부담 최소화
2. 검색 노출과 구독자 관리를 동시에 달성
3. 일관된 개인 브랜드 구축 지원

- **크리에이터 이코노미**: 2023년 기준 약 $250B, 연평균 20% 성장
- **이메일 마케팅 시장**: 2025년 약 $17.9B 예상
- **개인 브랜딩 수요**: 프리랜서/1인 기업 급증으로 지속 성장

## 🌟 비전 (Vision)

### 단기 목표 (1개월)
- 뉴스레터 발행 + 웹 아카이빙 기능 완성
- 기본 구독자 관리 기능

### 중기 목표 (6개월)
- SEO 최적화 시스템 구축
- Bio 기능 출시 (링크트리 대체)
- 분석 대시보드 고도화
- 수익화 기능 (유료 구독)

### 장기 목표 (1년)
- 글로벌 확장
- 멀티 브랜드 지원

## 💡 핵심 원칙

1. **심플함**: 복잡한 설정 없이 바로 시작
2. **통합성**: 한 곳에서 모든 것을 관리
3. **성장 지원**: 검색 노출을 통한 자연스러운 구독자 증가
4. **확장성**: 필요에 따라 기능 추가 가능
