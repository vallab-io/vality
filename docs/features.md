## 1️⃣ 뉴스레터 발송 + 웹 자동 아카이빙

### 기능 설명

뉴스레터를 작성하고 발행하면:
1. 구독자에게 이메일 발송
2. 웹에 자동으로 게시글 생성
3. RSS 피드 자동 업데이트

### 상세 기능

#### 1.1 에디터

```
┌─────────────────────────────────────────────────┐
│  📝 새 뉴스레터 작성                              │
├─────────────────────────────────────────────────┤
│  제목: [________________________]               │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │  WYSIWYG 에디터                          │   │
│  │  - 서식 (볼드, 이탤릭, 헤딩)              │   │
│  │  - 이미지 업로드                         │   │
│  │  - 링크 삽입                             │   │
│  │  - 코드 블록                             │   │
│  │  - 인용구                                │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│  [미리보기]  [임시저장]  [예약발행]  [발행하기]    │
└─────────────────────────────────────────────────┘
```

#### 1.2 발행 옵션

| 옵션 | 설명 |
|------|------|
| 즉시 발행 | 작성 즉시 이메일 발송 + 웹 게시 |
| 예약 발행 | 지정 시간에 자동 발행 |

#### 1.3 자동 URL 생성 규칙

```
https://vality.io/@{username}/{post-slug}
https://vality.io/@johndoe/weekly-tech-news-01
```

- slug는 제목에서 자동 생성
- 사용자가 직접 커스텀 가능
- 중복 시 자동으로 숫자 suffix 추가

### 기술적 요구사항

- [ ] 리치 텍스트 에디터 (TipTap 또는 Lexical)
- [ ] 이미지 업로드 및 최적화 (리사이징, WebP 변환)
- [ ] 이메일 발송 서비스 연동 (AWS SES, SendGrid, Resend)

## 2️⃣ SEO 최적화

### 자동 적용 항목

#### 2.1 메타 태그 자동 생성

```html
<!-- 기본 메타 태그 -->
<title>{게시글 제목} | {사용자명} - Vality</title>
<meta name="description" content="{게시글 요약 150자}">
<meta name="author" content="{사용자명}">
<meta name="keywords" content="{자동 추출 키워드}">

<!-- Open Graph -->
<meta property="og:title" content="{게시글 제목}">
<meta property="og:description" content="{게시글 요약}">
<meta property="og:image" content="{OG 이미지 URL}">
<meta property="og:url" content="{게시글 URL}">
<meta property="og:type" content="article">

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="{게시글 제목}">
<meta name="twitter:description" content="{게시글 요약}">
<meta name="twitter:image" content="{OG 이미지 URL}">
```

#### 2.2 구조화 데이터 (Schema.org)

```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "{게시글 제목}",
  "author": {
    "@type": "Person",
    "name": "{작성자 이름}",
    "url": "https://vality.io/@{username}"
  },
  "datePublished": "{발행일}",
  "dateModified": "{수정일}",
  "image": "{대표 이미지 URL}",
  "publisher": {
    "@type": "Organization",
    "name": "Vality",
    "logo": "https://vality.io/logo.png"
  },
  "description": "{게시글 요약}",
  "mainEntityOfPage": "{게시글 URL}"
}
```

#### 2.3 OG 이미지 자동 생성

```
┌─────────────────────────────────────┐
│                                     │
│  📰 게시글 제목이 여기에             │
│     표시됩니다                       │
│                                     │
│  @username                          │
│  vality.io                          │
│                                     │
└─────────────────────────────────────┘
```

- 제목 기반 자동 OG 이미지 생성
- 사용자 커스텀 이미지 업로드 가능
- 템플릿 선택 옵션 제공

### 기술적 요구사항

## 3️⃣ 구독자 관리

### 기능 상세

#### 3.1 구독 위젯

```
┌─────────────────────────────────────┐
│  📮 뉴스레터 구독하기                 │
│                                     │
│  [이메일 주소 입력          ] [구독] │
│                                     │
│  ✓ 매주 유익한 콘텐츠를 받아보세요    │
└─────────────────────────────────────┘
```

- 임베드 가능한 위젯
- 커스텀 스타일링 지원
- 더블 옵트인 지원

#### 3.2 구독자 대시보드

| 지표 | 설명 |
|------|------|
| 총 구독자 수 | 활성 구독자 카운트 |
| 신규 구독자 | 기간별 신규 가입자 |
| 이탈률 | 구독 취소율 |
| 오픈율 | 이메일 열람율 |
| 클릭율 | 링크 클릭율 |

#### 3.3 구독자 세그먼트

```
┌─ 구독자 그룹 ─────────────────────────┐
│                                       │
│  📁 전체 구독자 (1,234)               │
│  📁 활성 구독자 (1,100)               │
│  📁 VIP (50)                          │
│  📁 신규 (7일 이내) (89)              │
│  + 새 그룹 만들기                      │
│                                       │
└───────────────────────────────────────┘
```
