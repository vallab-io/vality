## 💰 서비스별 비용 상세

### 1. AWS SES (이메일 발송)

#### 비용 구조
- **EC2에서 발송 시**: 월 62,000건까지 **무료** ✅
- **일반 발송**: 1,000건당 **$0.10** (약 130원)

#### 사용량 예상

**MVP 단계:**
- 월 100명 신규 가입 → 인증 코드 200건
- 월 10개 뉴스레터 발행, 평균 50명 구독 → 500건
- **총 약 700건/월** → **무료** ✅

**성장 단계:**
- 월 1,000명 신규 가입 → 인증 코드 2,000건
- 월 50개 뉴스레터 발행, 평균 200명 구독 → 10,000건
- **총 약 12,000건/월** → **무료** ✅

**대량 발송 단계:**
- 월 5,000명 신규 가입 → 인증 코드 10,000건
- 월 200개 뉴스레터 발행, 평균 500명 구독 → 100,000건
- **총 약 110,000건/월**
  - EC2 발송 시: **$4.80/월** (약 6,240원) - 62,000건 무료 + 48,000건 유료
  - 일반 발송: **$11.00/월** (약 14,300원)

**결론**: MVP 및 성장 단계에서는 **무료**, 대량 발송 단계에서도 **$5-11/월**

### 2. AWS S3 (이미지 스토리지)

#### 비용 구조
- **스토리지**: GB당 **$0.023/월** (ap-northeast-2)
- **요청**: PUT 요청 1,000건당 **$0.005**
- **데이터 전송**: 아웃바운드 1GB당 **$0.09**

#### 사용량 예상

**MVP 단계:**
- 프로필 이미지: 사용자당 1개 (평균 200KB)
- 뉴스레터 이미지: 뉴스레터당 평균 3개 (평균 500KB)
- 월 100명 사용자, 50개 뉴스레터
  - 스토리지: 약 100MB → **$0.002/월**
  - 요청: 약 250건 → **$0.001/월**
  - 데이터 전송: 약 5GB → **$0.45/월**
  - **총 약 $0.45/월** (약 585원)

**성장 단계:**
- 월 1,000명 사용자, 200개 뉴스레터
  - 스토리지: 약 1GB → **$0.023/월**
  - 요청: 약 1,000건 → **$0.005/월**
  - 데이터 전송: 약 20GB → **$1.80/월**
  - **총 약 $1.83/월** (약 2,380원)

**대량 발송 단계:**
- 월 5,000명 사용자, 1,000개 뉴스레터
  - 스토리지: 약 5GB → **$0.12/월**
  - 요청: 약 5,000건 → **$0.025/월**
  - 데이터 전송: 약 100GB → **$9.00/월**
  - **총 약 $9.15/월** (약 11,900원)

**결론**: MVP 단계 **$0.50/월**, 성장 단계 **$2/월**, 대량 발송 **$10/월**
