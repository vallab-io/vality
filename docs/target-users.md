# 타겟 사용자 분석 (Target Users)

## 🎯 주요 타겟 그룹

### 1. 1인 창업자 (Solo Entrepreneurs)

#### 프로필
- **연령**: 25-45세
- **특징**: 자신의 전문성을 기반으로 사업 운영
- **예시**: 컨설턴트, 코치, 온라인 강사, 프리랜서 개발자

#### 니즈 (Needs)
- 브랜드 신뢰도 구축
- 고객과의 지속적인 관계 관리
- 시간 효율적인 마케팅 도구

#### Vality 제공 가치
| 니즈 | 제공 기능 | 기대 효과 |
|------|-----------|-----------|
| 브랜드 신뢰도 | 전문적인 웹 아카이브 | 검색 시 전문가 이미지 확보 |
| 고객 관리 | 구독자 관리 시스템 | CRM 역할 수행 |
| 마케팅 효율 | 원클릭 멀티채널 발행 | 시간 절약 |

#### 페르소나: 김민수 (32세, 스타트업 컨설턴트)
> "클라이언트들에게 정기적으로 인사이트를 공유하고 싶은데, 
> 블로그와 뉴스레터를 따로 관리하기엔 시간이 부족해요.
> 한 번에 처리할 수 있으면 좋겠어요."

---

### 2. 크리에이터 (Creators)

#### 프로필
- **연령**: 20-40세
- **특징**: 콘텐츠 제작이 주 수입원 또는 부업
- **예시**: 유튜버, 블로거, 인스타그래머, 틱톡커

#### 니즈 (Needs)
- 팬/구독자와의 친밀한 소통
- 플랫폼 의존도 분산 (알고리즘 리스크 헷지)
- 콘텐츠 아카이빙 및 재활용

#### Vality 제공 가치
| 니즈 | 제공 기능 | 기대 효과 |
|------|-----------|-----------|
| 팬 소통 | 팬레터/소식 전달 | 충성 팬층 구축 |
| 플랫폼 분산 | 자체 웹 아카이브 | 알고리즘 독립 |
| 콘텐츠 관리 | 통합 콘텐츠 허브 | 과거 콘텐츠 검색 노출 |

#### 페르소나: 이수진 (28세, 라이프스타일 유튜버)
> "유튜브 알고리즘이 변할 때마다 조회수가 롤러코스터예요.
> 이메일로 직접 연결된 팬들이 있으면 안정적일 것 같아요.
> 근데 블로그까지 따로 운영하긴 힘들어요."

---

### 3. 소규모 기업 (Small Businesses)

#### 프로필
- **규모**: 1-20명
- **특징**: 마케팅 전담 인력 부족
- **예시**: 로컬 비즈니스, 스타트업, 소호 쇼핑몰

#### 니즈 (Needs)
- 고객 공지 및 프로모션
- 브랜드 이미지 구축
- 비용 효율적인 마케팅

#### Vality 제공 가치
| 니즈 | 제공 기능 | 기대 효과 |
|------|-----------|-----------|
| 고객 소통 | 맞춤 뉴스레터 | 고객 리텐션 향상 |
| 브랜드 이미지 | 브랜드 페이지 | 전문적 인상 |
| 비용 효율 | 통합 플랫폼 | 툴 비용 절감 |

#### 페르소나: 최영희 (42세, 카페 운영)
> "단골 고객들에게 신메뉴나 이벤트를 알리고 싶은데,
> 인스타그램만으론 다 도달이 안 돼요.
> 이메일 보내면서 검색에도 노출되면 일석이조겠네요."

## 📊 타겟 우선순위

### Phase 1 (MVP)
1. **1인 창업자** - 얼리 어답터, 피드백 적극적
2. **크리에이터** - 바이럴 가능성, 입소문 마케팅

### Phase 2 (확장)
3. **글로벌 프리랜서** - Bio 기능 출시와 연계
4. **소규모 기업** - 유료 플랜 주요 타겟

---

## 🎯 공통 Pain Points

1. **멀티 플랫폼 관리 피로**: 여러 툴 동시 운영의 어려움
2. **SEO 진입 장벽**: 기술적 지식 부족
3. **시간 부족**: 콘텐츠 제작에만 집중하고 싶음
4. **브랜드 일관성**: 흩어진 채널로 인한 정체성 분산

## ✅ Vality의 해결책

```
"작성 한 번 → 발행 동시 → 모든 채널 완성"
```

- 시간 절약
- 자동 SEO 최적화
- 통합 브랜드 허브
