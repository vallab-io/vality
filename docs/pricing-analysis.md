## ⚠️ 리스크 분석

### 1. 대량 발송 사용자로 인한 손해

**문제:**
- Pro 플랜 ($29)으로 무제한 발송 시, 월 1,000,000통 이상 발송하면 손해 발생

**대응 방안:**

**옵션 1: Fair Use Policy (공정 사용 정책)**
- 무제한이지만 "합리적인 사용량" 제한
- 예: 월 500,000통까지는 무제한, 초과 시 추가 과금 또는 제한
- 장점: 명확한 제한, 예측 가능한 비용
- 단점: "무제한"이라는 표현과 모순

**옵션 2: 초과분 과금**
- 무제한이지만 월 500,000통 초과 시 추가 과금
- 예: 초과분 $0.10/1,000통
- 장점: 진정한 무제한 제공, 대량 사용자에게도 공정
- 단점: 복잡한 과금 구조

**옵션 3: 발송량 제한 (권장)**
- Pro 플랜을 "월 500,000통"으로 제한
- 초과 시 Enterprise 플랜 ($99/월) 안내
- 장점: 명확하고 예측 가능, 비용 관리 용이
- 단점: "무제한"이라는 표현 사용 불가

**옵션 4: 현재 제안 유지 + 모니터링**
- 현재 제안대로 진행하되, 대량 발송 사용자 모니터링
- 월 500,000통 이상 발송 시 개별 연락하여 Enterprise 플랜 제안
- 장점: 마케팅적으로 유리 ("무제한" 사용 가능)
- 단점: 초기 손해 가능성
