# 베타 단계 Pricing 페이지 구성안

## 🎯 목표

베타 서비스 단계에서 결제 시스템이 연결되기 전에:
1. **가격 정책 명확히 제시** - 사용자가 정식 출시 시 가격을 미리 알 수 있도록
2. **베타 혜택 강조** - 베타 사용자에게 특별 혜택 제공
3. **기대치 관리** - 정식 출시 일정 및 가격 정보 투명하게 공개
4. **차별화 포인트 강조** - 구독자 수 무제한, 발송 횟수 기반 과금

---

## 📐 페이지 구조

### 1. 헤더 섹션
- 제목: "가격 안내" / "Pricing"
- 설명: "베타 기간 중 모든 기능을 무료로 이용하세요"

### 2. 베타 혜택 배너 (현재 유지)
- Early Access 특별 혜택
- Founding Member 혜택
- "지금 가입하기" 버튼

### 3. 베타 기간 안내
- **"베타 기간 중 모든 기능 무료"** 강조
- 정식 출시 일정 안내 (예: "2025년 Q2 예정")
- 결제 시스템 출시 시 자동 전환 안내

### 4. 가격 구조 표시
- 가격 테이블 (결제 버튼 없이)
- "Coming Soon" 또는 "정식 출시 시 적용" 배지
- 이메일 발송 횟수 기반 과금 설명

### 5. 구독자 수 무제한 강조
- 차별화 포인트 강조
- 발송 횟수와 무관하게 구독자는 무제한

### 6. 애드온 정보 (선택사항)
- 애드온 목록 및 가격
- "정식 출시 시 제공 예정" 표시

### 7. FAQ 섹션
- 자주 묻는 질문
- 베타 기간 관련 질문
- 가격 정책 관련 질문

## 🎨 디자인 가이드

### 베타 배지
- "베타" 또는 "Beta" 배지
- "무료" 또는 "Free" 배지
- "Coming Soon" 배지 (결제 시스템)

### 색상
- 베타 혜택: Primary 색상 강조
- 가격 테이블: Neutral 색상
- "Coming Soon": Muted 색상

### 레이아웃
- 반응형 디자인
- 모바일 우선
- 카드 기반 레이아웃

## 📝 콘텐츠 예시

### 베타 기간 안내
```
베타 기간 중 모든 기능을 무료로 이용하실 수 있습니다.
정식 출시 시(2025년 Q2 예정) 가격 정책이 적용되며,
베타 사용자는 Founding Member 혜택을 받으실 수 있습니다.
```

### 가격 구조
```
정식 출시 시 적용될 가격입니다.
베타 기간 중에는 모든 기능을 무료로 이용하실 수 있습니다.
```

### FAQ
- Q: 베타 기간은 언제까지인가요?
- A: 정식 출시 시(2025년 Q2 예정)까지입니다.

- Q: 베타 사용자는 정식 출시 후에도 무료인가요?
- A: 베타 사용자는 Founding Member 혜택으로 3개월간 무료 이용이 가능합니다.

- Q: 구독자 수 제한이 있나요?
- A: 아니요. 구독자 수는 무제한입니다. 발송 횟수만 과금됩니다.
